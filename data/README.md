# Datos del proyecto — CallMeMaybe

Contiene los datasets originales usados en el análisis de eficiencia de operadores.

## Archivos

- `telecom_dataset_us.csv` — Registros de llamadas por operador (dirección, duración, llamadas perdidas, tiempo de espera)
- `telecom_clients_us.csv` — Información de clientes (plan tarifario, fecha de registro)

## Nota

Los datos corresponden al período 2019 y fueron proporcionados como parte del proyecto de análisis de telefonía virtual CallMeMaybe.
